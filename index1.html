GENERALIDADES DE CSS I

PARTES DE CSS

La estructura de css, tiene selector propiedad y valor

El selector es el nombre que tiene como classe (class) la etiqueta de html

La propiedad se refiere a lo que vamos a manipular de esa etiqueta HTML, ya sea el color, el tamaño o la letra

El valor hace referencia a especificamente, que va a cambiar si es el tamaño de letra por ejemplo 9px

SELECTORES

SELECTOR UNIVERSAL(*): Selecciona todo en el HTML

SELECTOR DE TIPO(H1,BODY,ETC):

Selecciona todas las etiquetas de ese tipo, por ejemplo si aplicas la propiedad a
la etiqueta h2 todos los h2 seguiran la propiedad

SELECTOR POR CLASES:

Se agrega el atributo class a la etiqueta HTML y se esxoge como selector en el CSS

En el HTML se le agrega la palabra class="" y dentro de las comillas se da el nombre de clase

En el CSS se le pone la clase como selector y se le da los valores de propiedad y valor

En el browser se ve reflejado el cambio echo en css

SELECTOR DE ID

Se pone en HTML como id="aqui va el nombre del id"

los Id son unicos, no se pueden repetir dentro del archivo

Se pone en CSS como el selector

Se refleja el cambio en el browser

SELECTOR COMO ATRIBUTO

Se puede usar un atributo (inventado puede ser cualquiera)

Se usa ese mismo atributo entre corchetes para el CSS

El resultado en el browser seria que los cambios se aplican

SELECTOR COMO ELEMENTOS HIJO

Se puede indicar la ruta del elemento hijo

En el html, se crea la estructura el elemento padre es el div, los elementos hijos
son los h4 y el p

Se le indica la ruta desde el CSS, para indicar que debe cambiar

El cambio se ve reflejado en el Browser


PROPIEDAD

Propiedad: Es la característica o aspecto del elemento que deseas modificar. Puede ser color para el
color del texto, background-color para el color de fondo, font-size para el tamaño de fuente, etc.
