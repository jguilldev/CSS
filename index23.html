PSEUDOELEMENTOS I

::before

Este pseudolemento se usa para insertar contenido antes del contenido real de un elemento, su uso principal
es agregar iconos decoraciones o efectos sin cambiar el HTML, esta propiedad se hereda a los elementos hijos
de un contenedor, es necesario, que este elemento use la propiedad content, incluso aunque esta ultima propiedad este vacia

::after

Este elemento se usa para insertar contenido despues delcontenido real del elemento, al igual que ::after
agrega decoraciones, iconos o elementos visuales, sin realizar cambios en el HTML, para usar este elemento,
es necesario, que este elemento use la propiedad content, incluso aunque esta ultima propiedad este vacia
