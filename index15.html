DISPLAY

Ya vimos que la propiedad display controla la forma como se comporta y se ubica un elemento con respecto al
view-port (espacio de visualizacion) y el flujo del documento y sus elementos

Ya vimos las dos propiedades o valores de display que son BLOCK e IN-LINE
puedes hacer click a cada una de las palabras anteriores (en verde) para ver la explicacion, ahora veremos las demas:

Inline-block

Estos elementos se comportan como inline, pero se les puede aplicar widht y height

table

Se comporta similar a una tabla de HTML, sus elementos hijos se asemejaran a el contenido de la tabla







inline-table

Funciona similar al anterior, asemejandose a una tabla pero ocupando todo el espacio disponible como
comportaria un elemento en linea.







list-item

Se comporta como un item lista de HTML < li > < /li > e incluye un marcador o viñeta por defecto
exactamente como un elemento li de HTML

table-cell

Se refiere al comportamiento de una celda en una tabla







table-row

Se refiere al comportamiento de una fila en una tabla







table-column

Se refiere al comportamiento de una columna en una tabla,
sin embargo sus efectos no son tan visibles, como si ocurre con otras propiedades

La propiedad table-column está diseñada para ser utilizada junto con la estructura de una tabla real y solo debe
aplicarse a elementos como < col > o < colgroup > dentro de una tabla.



none

No se muestra en ningun momento y no ocupa espacio en el documento es como si saliera del flujo
muy util para eliminar elementos momentaneamente al hacer pruebas.









con display block si aparece en el flujo normal del elemento
