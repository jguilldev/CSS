GENERALIDADES DE CSS

PARTES DE CSS

La estructura de css, tiene selector propiedad y valor

El selector es el nombre que tiene como classe (class) la etiqueta de html

La propiedad se refiere a lo que vamos a manipular de esa etiqueta HTML, ya sea el color, el tamaño o la letra

El valor hace referencia a especificamente, que va a cambiar si es el tamaño de letra por ejemplo 9px

SELECTORES

SELECTOR UNIVERSAL(*): Selecciona todo en el HTML

SELECTOR DE TIPO(H1,BODY,ETC): Selecciona todas las etiquetas de ese tipo, por ejemplo si aplicas la propiedad a
la etiqueta h2 todos los h2 seguiran la propiedad

SELECTOR POR CLASES: Se agrega el atributo class a la etiqueta HTML y se esxoge como selector en el CSS

En el HTML se le agrega la palabra class="" y dentro de las comillas se da el nombre de clase

En el CSS se le pone la clase como selector y se le da los valores de propiedad y valor

En el browser se ve reflejado el cambio echo en css
